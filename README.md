# todolist-express

代码目录分三个部分：

- todo-server，用express写的5个接口
- todo-web-react，用react和antd写的前端页面
- todo-web-vue3，用vue3和element-plus写的前端页面

## 启动该项目

1. todo-server

    ```bash
    npm run start
    ```

    启动接口，端口3333

    > PS:请修改数据库的用户和密码，该服务启动在作者的本地mysql

2. todo-web

    ```bash
    npm run start
    ```

    启动页面
